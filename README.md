## Deskripsi Tugas

Repository ini berisi tugas praktikum mengenai struktur blok prosedural pada MySQL menggunakan stored procedure.

Materi yang dikerjakan meliputi:

* Bagian A : Identitas Mahasiswa
* Bagian B : Validasi Data Akademik
* Bagian C : Ringkasan Kelayakan KRS
* Bagian D : Analisis Dua Mahasiswa
* Skenario Pengujian Valid dan Tidak Valid

## Cara Menjalankan Program

1. Buka phpMyAdmin / MySQL
2. Import atau jalankan file `program.sql`
3. Jalankan procedure dengan perintah `CALL nama_procedure();`
